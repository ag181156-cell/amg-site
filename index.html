Atom AI
Builder & Product Lead · Apr 2026 · Maracaibo —
Creador & Product Lead · Abr 2026 · Maracaibo —
Claude API Supabase React + TypeScript 9 DB tables 4 AI Edge Functions
The Problem
El Problema
Sales teams in Venezuela operate with WhatsApp, Excel, and notebooks. No pipeline visibility, prospects lost for lack of follow-up, no real-time data, no way for managers to measure their team. I built Atom AI to eliminate all of that.
Los equipos de ventas en Venezuela operan con WhatsApp, Excel y cuadernos. Sin visibilidad del pipeline, prospectos perdidos por falta de seguimiento, sin datos en tiempo real. Construí Atom AI para eliminar todo eso.
What I Built
Lo que Construí
A full commercial operations platform — CRM with anti-collision (90-day attribution, 30-day auto-release), AI assistant powered by Claude that generates personalized pitches, analyzes prospects, and suggests next actions. Three differentiated role dashboards: Admin, Manager, and Seller. Research center, team management, real-time KPIs, commission calculator, and internal communication system.
Una plataforma completa de operaciones comerciales — CRM con anti-colisión (atribución de 90 días, liberación automática a 30 días), asistente de IA con Claude que genera pitches personalizados, analiza prospectos y sugiere la próxima acción. Tres dashboards diferenciados por rol: Admin, Manager y Vendedor. Centro de investigación, gestión de equipos, KPIs en tiempo real, calculadora de comisiones y comunicación interna.
The Stack
El Stack
React + TypeScript frontend. Supabase — PostgreSQL with Row Level Security on all 9 tables, Auth, Edge Functions, and Storage. Claude API (Anthropic) for 4 AI functions.
Frontend en React + TypeScript. Supabase — PostgreSQL con Row Level Security en las 9 tablas, Auth, Edge Functions y Storage. Claude API (Anthropic) para 4 funciones de IA.

Atom AI is Rush Studio's internal tool — but it's also proof of concept. If we can build this for ourselves, we can build it for any sales team in Latin America. v1.0 launched April 2026, currently in active use by the Rush Studio commercial team in Maracaibo.

Atom AI es la herramienta interna de Rush Studio — pero también es prueba de concepto. Si lo podemos construir para nosotros, lo podemos construir para cualquier equipo comercial en Latinoamérica. v1.0 lanzada en abril 2026, en uso activo por el equipo comercial de Rush Studio en Maracaibo.
